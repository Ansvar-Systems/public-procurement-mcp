# Changelog

All notable changes to the Public Procurement MCP Server will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-25

### Added

- Initial scaffold with 19 MCP tool stubs
- Database schema for legal content, reference data, and award intelligence
- SQLite adapter with parameterized query helpers
- MCP server with stdio and HTTP transports
- Vercel serverless handlers
- Docker multi-stage build
- CI/CD workflows (build, test, CodeQL, Gitleaks, Trivy)
- Open-source documentation (LICENSE, SECURITY, DISCLAIMER, PRIVACY, CONTRIBUTING)
